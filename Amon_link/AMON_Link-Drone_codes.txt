AMON Link - Drone codes
Code ID:
00 - NULL
01 - Link setup
02 - Link error
03 - Pair
04 - Link response
11 - Amon setup
12 - Amon error
13 - Amon parameters
14 - Amon response
20 - Amon arm
21 - Amon fly data
30 - Amon land
